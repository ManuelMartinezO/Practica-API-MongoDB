## API con MongoDB
Una Api construida en javascript, mongoDB como base de datos, funcion de cargar y descargar archivos.
### Endpoints
```
Crear / Subir       POST    /api/games/upload
Listar Todos        GET     /api/games
Obtener Uno         GET     /api/games/:id
Actualizar          PUT     /api/games/:id
Eliminar            DELETE  /api/games/:id
Descargar Archivo   GET     /api/games/:id/download
```
